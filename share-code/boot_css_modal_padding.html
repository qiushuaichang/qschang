td点击事件+固定表头影响页面样式

自己做的表头，在有多列的表格中，选中多个td然后向右拉动，表格内容向右滑动，鼠标抬起的时候，会引起页面乱，这时候很有可能就是给td添加点击事件的原因，此时把td的事件给tr就行了

两个方法同时监听同一个对象

这是稍微偏浏览器底层的，两个方法有可能会同时监听到这一个对象，然后“资源竞争”，影响到其他的方面或者整体的样式。这样可以用“锁”去解决。原理就是执行一个方法的时候，就false另一个方法(进行判断，让这个方法不执行)
